      https://github.com/Gael-Lopes-Da-Silva/YellowedHelix

Description
------------------------------------------------------------------

Yellowed is a simple material dark theme wich focus his syntax
color on the golden yellow. Its syntax is inspired by the
gruber-darker theme for emacs, but changed a bit.

Installation
------------------------------------------------------------------
To install the theme you just need to put the `yellowed.toml`
file into the themes folder of your Helix config folder. This
one should be on `~/.config/helix/themes` on Linux or
`~/AppData/Roaming/helix/themes` on Windows. See
https://docs.helix-editor.com/themes.html for more informations.
